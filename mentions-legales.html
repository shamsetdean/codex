03

Domaine public — Base légale

Toutes les œuvres présentées sur Codex sont dans le domaine public conformément à l'article L123-1 du Code de la propriété intellectuelle (CPI) français, qui prévoit l'extinction des droits patrimoniaux 70 ans après le décès de l'auteur (post mortem auctoris, ou p.m.a.).

Pour les œuvres en collaboration, la protection expire 70 ans après le décès du dernier auteur survivant. Pour les œuvres anonymes ou pseudonymes, le délai court à compter de la publication.

Texte de référence : Art. L123-1 CPI — "L'auteur jouit, sa vie durant, du droit exclusif d'exploiter son œuvre sous quelque forme que ce soit et d'en tirer un profit pécuniaire. Au décès de l'auteur, ce droit persiste au bénéfice de ses ayants droit pendant l'année civile en cours et les soixante-dix années qui suivent."

⚠ Attention aux traductions et éditions récentes : une traduction récente d'une œuvre ancienne peut être protégée, même si l'œuvre originale est dans le domaine public. Codex utilise exclusivement des traductions elles-mêmes tombées dans le domaine public (ex. : traductions de François-Victor Hugo, Louis Viardot, Leconte de Lisle, Lamennais).

Domaine public aux États-Unis : Le Projet Gutenberg applique la législation américaine (œuvres publiées avant 1928). Toutes les œuvres référencées sur Codex sont libres de droits en France et aux États-Unis.

04

Accès aux textes — Modèle technique

Codex n'héberge aucun fichier de texte. Les textes affichés dans le lecteur intégré sont chargés en temps réel depuis le Projet Gutenberg (gutenberg.org) via un proxy CORS (corsproxy.io). Ce modèle est légalement propre : Codex agit comme un agrégateur de liens, non comme un hébergeur.

En cas d'indisponibilité du proxy ou de Gutenberg, l'utilisateur est redirigé directement vers la page source sur gutenberg.org. Les liens de téléchargement (EPUB, TXT) pointent également directement vers les serveurs de Gutenberg.

Source primaire : Project Gutenberg — organisation à but non lucratif fondée en 1971, plus de 70 000 livres libres de droits.

06

Photos des auteurs

Codex affiche des photos d'auteurs uniquement lorsque leur licence est confirmée comme relevant du domaine public ou étant compatible avec une utilisation commerciale libre.

Les photos sont chargées en temps réel depuis Wikimedia Commons via l'API REST de Wikipédia. Avant affichage, le code vérifie la licence de chaque image via l'API MediaWiki (champ extmetadata.LicenseShortName). Une image n'est affichée que si sa licence contient explicitement les termes PD, Public Domain ou CC0.

Codex n'héberge aucune image. Toutes les photos sont chargées directement depuis les serveurs de Wikimedia Commons. Si une image affichée vous semble incorrectement licenciée, contactez-nous à shamsetdean@gmail.com.

Les portraits de personnalités décédées avant 1900 sont dans le domaine public en vertu de l'ancienneté des clichés photographiques. Certaines images proviennent de peintures ou gravures elles-mêmes dans le domaine public.
